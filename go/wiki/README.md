## 贡献

- 这个维基可被拥有 Github 账号的 Go 社区的任意成员编辑。
- 如果你想要新增一个页面，请首先在 [Go issue 跟踪页面](https://github.com/golang/go/issues) 打开一个 issue，以前缀 “wiki” 开头来提议新增的内容。清楚地说明为什么这个内容不适用任何现有的页面。
- 因为维基页面的重命名会破坏外部链接，请在重命名或删除任何维基页面之前打开一个 issue。
